Pollfish SDK for Android
------------------------

The is the Pollfish SDK for Android.

Requirements:
- A Pollfish API Key for your app (http://www.pollfish.com)
- Setup Google Play Services for your app (http://developer.android.com/google/play-services/setup.html) 
- Integrate Pollfish .jar or .aar library in your project
- Runtime of Android 2.3.3 (10) or later.
- Please see documentation for handling orientation changes properly
- Please see Sample project if you wish to incentivise your users for completing a survey

The SDK allows the completion of surveys through a Pollfish enabled app


More information can be found in the documentation online at https://www.pollfish.com/android
